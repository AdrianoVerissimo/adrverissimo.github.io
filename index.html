Escapenaut

Available for Windows, Linux and MacOS

The spaceship has been attacked and activated its self-defense system! Help the astronaut escape the ship, while avoiding dangerous traps and solving puzzles!

This game was created for NES Jam, where we should create a game based on the old Nintendo, with the resolution of 256x240 and the color palette of the console.

Control the girl and pick up apples to get the highest score possible! Don't catch the bombs, and beware of the birds: they will hinder the girl to pick up the apples!

This game was developed for the 4th Games Production Marathon, from the course Produção de Jogos, with a duration of 12 weaks.

  • Game Release Date: 2016/11/25
  • My participation: Programming, Art and Audio

Alfred in the Paint World

Available for Web, Windows, MacOS and Linux

Click here to play


Alfred is lost in Paint World and needs to find his way back home.

Help the chameleon find the exit in this 2D platform game by jumping through holes, pushing boxes and avoiding dangerous traps!

This game was made for the MS Paint Jam, a Game Jam that we should create a game that remembers the feeling of the MS Paint.

  • Game Release Date: 2017/08/31
  • My Participation: Programming and Art

  • Used Software:
    • Game Engine: Unity using C#
